## Installation

Stable releases can be installed from PyPI.

```bash
pip install compas_pr2d
```

To install the latest version for development, do:

```bash
git clone https://github.com/BRG-research/compas_pr2d.git
cd compas_pr2d
pip install -e ".[dev]"
```
